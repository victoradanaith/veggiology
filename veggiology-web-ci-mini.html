03 — Logo Quick Rules

Text-only summary — actual logo artwork (.ai / .eps / .png) lives in the Drive folder VGL_Artwork > 01_Brand Identity, not reproduced here.

Clear space & minimum size

Clear space = 1.5X around the Primary Logotype (X = width of the letter "I"). Minimum height: 0.4cm for Primary Logotype, 0.2cm for Secondary Logotype — switch to Secondary below 0.4cm. Symbol clear space = 0.5× its own size, min height 0.4cm.

3 color modes

Color Positive (Soil Black) on light backgrounds. Color Negative (Warm Neutral) on dark backgrounds. Grayscale (black or white only) for 1-color print.

Symbol meaning

Plant (ingredients & sustainability) + Water (the product itself) + Heart (honesty and care) — the three combine to represent the brand's promise.

Don't
  • Recolor outside approved modes
  • Rotate, distort, or outline it
  • Add shadows or effects
  • Place on low-contrast backgrounds
  • Alter the typeface or structure
  • Print an unapproved tagline

04 — CSS Design Tokens

Copy-paste ready. Corrects the earlier scraped file, which had aliased the cream background to plain white and used an unofficial green.

:root{
  /* Core */
  --vgl-warm-neutral: #F7F1E3;
  --vgl-soil-black:   #000000;
  --vgl-veggie-green: #0E2517;

  /* Supporting — Tint / Pure / Tone */
  --vgl-blue-tint:#AFDCEB;   --vgl-blue-pure:#82CDE6;   --vgl-blue-tone:#5ABEDC;
  --vgl-green-tint:#C3D7A5;  --vgl-green-pure:#73C378;  --vgl-green-tone:#87B978;
  --vgl-yellow-tint:#FFDC91; --vgl-yellow-pure:#FACD55; --vgl-yellow-tone:#FFC300;
  --vgl-pink-tint:#F5CDD2;   --vgl-pink-pure:#F5A5B4;   --vgl-pink-tone:#F07DA0;
  --vgl-red-tint:#FAB9A0;    --vgl-red-pure:#F58773;    --vgl-red-tone:#EB6446;

  /* Fonts — replace licensed placeholders with real files when available */
  --font-display-en: "PP Woodland", "Fraunces", Georgia, serif;
  --font-accent-en:  "TT Drugs", "Archivo Black", sans-serif;
  --font-body-en:    "Work Sans", Arial, sans-serif;
  --font-display-th: "Kulachat Serif", "Noto Serif Thai", serif;
  --font-body-th:    "Prompt", sans-serif;
  --font-small-th:   "Bai Jamjuree", sans-serif;
}
